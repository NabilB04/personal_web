Projects

Kompos-In

Kompos-In is a web-based smart waste management system I developed to support eco-friendly household composting. The platform connects administrators and residents through a real-time dashboard that tracks trash volume and pickup schedules. Built using Laravel and Tailwind CSS, the system integrates with IoT sensors to detect when bins are full and uses Leaflet.js to visualize optimized pickup routes. Residents can monitor their waste contributions, receive notifications, and see how much compost they’ve helped produce. Admins can manage articles, users, and waste data effortlessly, while a role-based login system ensures security and tailored experiences for each user type. This project reflects my interest in sustainability and how technology can play a role in driving environmental change.

LaporAja

LaporAja is a full-stack web application designed to streamline public complaint handling for local government simulation. The system enables citizens to report issues such as infrastructure damage or public service problems, and allows administrators to manage reports with status updates and responses. Built using Laravel, Tailwind CSS, and Cloudinary for image handling, the app features multi-role authentication, email verification, and real-time dashboards. Citizens can submit complaints with geolocation and image evidence, while admins can review, respond, and track statistics. The system also supports data export (PDF/Excel) and integrates SweetAlert for enhanced UX interactions.

TaniAttire

A desktop application developed using C# and Windows Forms to manage product inventory, rentals, and sales for a fashion retail business. The system includes modules for product and stock management, rental tracking, and transaction reporting. It connects to a PostgreSQL database using Npgsql for robust data handling.
